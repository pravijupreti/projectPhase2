# Project Phase 2 - AI/ML Development Engine

This is the main engine repository containing development scripts, prototypes, and testing code for Project Phase 2.

## Structure
- **Windows/** - PowerShell automation scripts for Windows environment
- **Linux/** - Shell scripts for Linux distributions
- **app.py** - Main application code (currently mixed with testing)
- **Untitled.ipynb** - Jupyter notebooks for experimentation

## Status: Prototype Phase
Currently in active development where testing code and app code coexist. Git integration and cross-platform support are still in progress. After this phase, business logic and engine will be separated into dedicated modules.
